This is the code for the paper: "Temporal local clustering coefficient uncovers the hidden pattern in temporal networks"


-compute.py compute the TC/C for the empirical dataset
-drawfig.py draw TC/C plot for the empirical dataset
-compute_and_draw_TCdt.py compute and draw the TC in different time windows for the empirical dataset
-compute_and_draw_shuffled.py compute and draw the TC for randomized dataset
-simulation.py compute and draw the TC/C for simulation model
